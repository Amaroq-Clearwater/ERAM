How to install on:

x86
---
1. Copy eramnt.sys to \Windows\System32\drivers.
2. Import eramnt.reg to the Registry.
3. Restart

x64
---
1. Copy eramnt64.sys to \Windows\System32\drivers
2. Import eram64.reg to the Registry.
3. Restart

After installing, the RAM Disk will be available as the R: drive with a size of 996 MB.
You can use the included eramnt.cpl (x86) / eram64.cpl (x64) to change the size, etc. of the RAM Disk.
